# hcva-plus-plus/input_data

The input data should be in the format dd-mm-yyyy-type-source.csv, where type means:

- principles: the principle preference data only.
- agent: the agent data for a dataset that contains their personal value preferences and action judgements
- personals: the agent data for a dataset that contains their personal value preferences  
- actions: the agent data for a dataset that contains their aciton judgements
- raw: The agent data for a dataset that is not formatted as a value system, but rather the raw inputs.
